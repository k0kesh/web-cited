Logo concepts.

Ten square SVG marks, all tied to the practice — citations, answer extraction, generative-engine visibility, voice/AEO, wayfinding. Each preview shows the mark on paper and inverted on ink, plus a sample lock-up beside the wordmark.

01 · Citation Brackets
Web Cited

Citation Brackets

The strongest single-glyph metaphor for "cited" — square brackets enclosing a single source. Reads instantly as editorial / reference material.

02 · Answer-Extract Quote
Web Cited

Answer-Extract Quote

Open-quote glyphs above an extracted answer block — directly visualizes how generative engines lift content from a source page.

03 · Citation Rays
Web Cited

Citation Rays

Knowledge-graph node radiating in every direction — your brand as the central entity that engines reference outward from.

04 · Footnote Mark
Web Cited

Footnote Mark

A serif asterisk paired with an orange "1" superscript — the literal citation marker in print and academic writing.

05 · Crawl Path
Web Cited

Crawl Path

Three connected nodes climbing — site → analysis → cited destination. Quietly evokes a directed crawl traversal.

06 · Voice Waveform
Web Cited

Voice Waveform

Five vertical bars in a peak — a visual nod to AEO, voice queries, and the spoken-answer surfaces (Alexa, Siri, Assistant).

07 · North Star
Web Cited

North Star

Wayfinding mark — a four-point compass star with an orange core. Reads as direction, navigation, the "where buyers actually look."

08 · Magnifier Dot
Web Cited

Magnifier Dot

Direct-meaning search lens — but rendered as a chunky editorial mark, not a thin Bootstrap icon. Orange dot is the citation found.

09 · Pilcrow
Web Cited

Pilcrow

The paragraph mark — a uniquely editorial glyph. Says "we look at the prose" without leaning on a generic search/chart icon.

10 · Caret Query
Web Cited

Caret Query

Angle brackets enclosing a query cursor — code-aware, reads as both technical SEO and the prompt input field on every LLM.
